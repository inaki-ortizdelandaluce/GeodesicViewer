Metric



The database of space-time metrics is taken from the Motion4D library. Currently, the following metrics are implemented:
  • Minkowski
  • Schwarzschild
  • Alcubierre
  • Barriola-Vilenkin
  • Bertotti-Kasner
  • Bessel gravitational wave
  • Cosmic string
  • Ernst
  • Extreme Reissner Nordstrom dihole
  • Friedman non empty
  • Gödel
  • Halilsoy wave
  • Janis-Newman-Winicour
  • Kasner
  • Kerr
  • Kottler
  • Morris-Thorne.
  • PetrovTypeD AI-AIII, BI-BII, C
  • Reissner-Nordstrom.
  • DeSitter
  • StraightSpinningString
  • Sultana-Dyer black hole
  • TaubNUT
  • Teo wormhole
Depending on the metric, there will be a list of all parameters. For the Kerr metric, e.g., we can adjust the angular momentum as well as the mass of the black hole.

Integrator



Implemented integrators:
  • Standard Runge-Kutta fourth-order.
  • Standard Bulirsch-Stoer.
  • Integrators from the GNU Scientific Library (GSL).

Parameters:
Name Description
draw with lines Connect the points of the geodesic with lines.
stepsize controlled (De)Activate stepsize control for the integrator.
max points Maximum number of points to be calculated for a geodesic.
step size Step size for the integrator.
constr. eps epsilon value for the constraint equation.
eps abs absolute epsilon
eps rel relative epsilon
max step maximum step size (Bulirsch Stoer only)


Constants

The fundamental constants like the speed of light, Newton's gravitational constant or the dielectric permittivity can be given in SI or geometrical units.
